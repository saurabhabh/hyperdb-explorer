# Introduction

A small application which you can use to explore/browse your Hyper db. 

Currently runs with:

- Angular v7.2.0
- Electron v4.0.0
- Electron Builder v20.28.1
- HyperDB v3.5.0

With this sample, you can :

- Filter
- Browse
- View Details of a Node via Key.
